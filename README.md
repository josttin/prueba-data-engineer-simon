Prueba Técnica: Ingeniero de Datos Junior - Procesamiento de Flota

Este proyecto resuelve el desafío de procesar los datos de viajes de taxis de NYC para generar dos reportes clave: la última ubicación conocida por vehículo y el conteo de viajes por hora.


1. Herramientas Elegidas

Se seleccionó el ecosistema de Python por su versatilidad, velocidad de desarrollo y su estándar en la industria de la ingeniería de datos.

Lenguaje: Python 3.x
Librerías:
Pandas: Motor principal para la manipulación y transformación de DataFrames.
PyArrow: Utilizado para la lectura eficiente de archivos en formato Parquet.

2. Instalación de Dependencias

Se recomienda usar un entorno virtual (`venv`).

1.  Asegúrate de que Python esté instalado.
2.  Navega a la carpeta del proyecto y activa el entorno virtual (ej. `.\venv\Scripts\activate`).
3.  Instala las dependencias listadas en `requirements.txt`:

```bash
pip install -r requirements.txt
